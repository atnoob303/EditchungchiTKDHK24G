# 🎓 Chứng chỉ Save the Children — Trình chỉnh sửa

Chỉnh sửa tên & ngày tháng trực tiếp trên PDF chứng chỉ gốc, hoàn toàn trên trình duyệt.

## 🚀 Dùng ngay trên GitHub Pages

1. Fork repo này
2. Vào **Settings → Pages → Source: main branch / root**
3. Truy cập `https://TÊN_BẠN.github.io/TÊN_REPO/`

## 💻 Chạy local

Chỉ cần mở file `index.html` trong trình duyệt — không cần cài gì!

## ✨ Tính năng

- Nhập tên → xem trước live ngay trên trang
- Chọn ngày bất kỳ hoặc dùng ngày hôm nay  
- Xem trước PDF trong trang trước khi tải
- Tải về file `chung_chi_TÊN.pdf`
- Font `FreeMonoOblique` đúng 100% với bản gốc
- Màu đỏ `#D11000` khớp chính xác
- Không cần server, không cần backend
